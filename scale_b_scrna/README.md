# Scale B — Single-cell RNA-seq (GSE234904)

Single-cell RNA-seq analysis of peripheral neutrophils from a COVID-19 ARDS cohort, focused on the HIF1A pathway and stratified by 28-day extubation outcome.

## Dataset

- **Accession**: NCBI GEO **GSE234904**
- **Publication**: Ito H, Ishikawa M, Yoshimura J, Liu Y, Sakakibara S, Sugihara F, Matsumoto H, Hirata H, Ogura H, Oda J, Okuzaki D. *Neutrophil gene expression in COVID-19 patients with acute respiratory distress syndrome.* Front Immunol. 2025;16:1620745. doi:10.3389/fimmu.2025.1620745.
- **Original analysis code**: https://github.com/amufaamo/covid-neutrophil-ards

## Cohort composition

The original GSE234904 dataset comprises:
- **5 COVID-19 ARDS patients**:
  - Cases 1, 2: weaned within 28 days (VW)
  - Cases 3, 4, 5: not weaned within 28 days (NVW)
- **6 healthy controls**, multiplexed via TotalSeq-C hashtag antibodies into a single GSM (GSM7476346)

In the present Scale B scRNA analysis, **25,773 peripheral neutrophils** were retained after QC.

## Scripts (four-phase pipeline)

| Script | Phase | Function |
|---|---|---|
| `scripts/17_scale_b_scRNA_phase1.py` | 1 | Load 10x feature-barcode matrices, demultiplex hashtag antibodies, initial QC |
| `scripts/18_scale_b_scRNA_phase2.py` | 2 | Per-sample filtering, normalization, HVG selection, PCA |
| `scripts/19_scale_b_scRNA_harmony.py` | 3 | Harmony integration across donors, UMAP, Leiden clustering |
| `scripts/20_scale_b_scRNA_phase3.py` | 4 | HIF1A pathway scoring, weaning-stratified DEG analysis, cluster characterization |

## Data access

Raw 10x Genomics feature-barcode matrices (`*_raw_feature.h5`) and Cell Ranger outputs are available via GEO. See [`data/README.md`](data/README.md) for the full download procedure.

The processed `.h5ad` files (merged_raw, merged_processed, merged_harmony, merged_harmony_scored — each 90 MB to 1.8 GB) are **not redistributed here** but can be regenerated by running the four-phase pipeline on the GEO raw files. The fully scored object (`merged_harmony_scored.h5ad`) is available on request from the corresponding author for cases where users wish to skip the pipeline.

## Results in this repository

- `results/sample_summary.csv` — per-sample QC summary
- `results/scale_b_scRNA_results.csv` — pathway scores, cluster assignments, weaning stratification
- `results/DEG_NonWeaned_vs_Weaned.csv` — differential expression analysis (non-weaned vs weaned)

Figure outputs (UMAP, HIF1A pathway score distributions, cluster heatmaps, DEG barplots) are copied to `figures/main/` and `figures/supplementary/`.

## Key finding

A HIF1A-high neutrophil cluster (Leiden cluster 2) is enriched in non-weaned ARDS patients, with all 3 non-weaned patients showing higher mean HIF1A pathway scores than both weaned patients (Cohen's d=+0.31 at cluster level, p≈10⁻⁷⁰ after pseudoreplication-aware test).
